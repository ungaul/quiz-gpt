How to use?

Download the executable of ClipGPT.

Go to OpenAI and sign up for an account.

In API Keys, create a new API key. You have to add 5$ to your account to use the API, but each request cost less than 0.01$.

On Windows, go to Edit the System Environment Variables, then Environment Variables.

Click on New... then set the variable name to OPENAI_API_KEY and the value to your OpenAI API key, then click OK and close the window.

(Optional) If you want to use a specific model, add a variable called GPT_MODEL and set the value to the model you want (please refer to here for the models' names). You can also specify your requests by adding a variable called GPT_PROMPT and set the value to what you want to work on. Finally, set CONTINUOUS_CONVERSATION to true if you want to have a continuous conversation with the AI (cost more, but remembers your previous messages).

Click on the executable to get started!
